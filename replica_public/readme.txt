This folder contains the replication files for all the figures in the main
text and most of the figures in the Online Appendix.

To replicate a Figure, open the file fig_XX, where XX is the number of the
figure as it appears in the paper, and run it. The opening lines of file
fig_XX also tell you the URL where the figure of interest gets stored.

The CrossBorder Capital Ltd. (CBC) dataset used in several of the exercises
is proprietary and we are not authorised to share it. However, most of the
results can be replicated without these series. Figure 5 (The Commodity
Price Channel) does not require CBC data. Figure 1 does, but you will 
find in the replication folder the file "fig_1_public.m" which produces
a similar plot as in the main text without including the CBC data.

The code has been tested with Matlab R2024a.
